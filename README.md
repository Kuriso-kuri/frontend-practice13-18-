# Приложение заметок — Практические занятия 13–18

Практическая работа №13 — Service Worker

В рамках данной работы было разработано веб-приложение для управления заметками с базовым офлайн-функционалом. Реализована регистрация Service Worker, кэширование статических ресурсов при установке и перехват сетевых запросов. Заметки сохраняются в localStorage, что позволяет работать с ними без подключения к интернету.

Практическая работа №14 — Web App Manifest

В данной работе приложение было превращено в прогрессивное веб-приложение (PWA) с помощью файла manifest.json. Добавлены иконки разных размеров, мета-теги для интеграции с мобильными платформами и поддержка установки приложения на устройство. Service Worker был обновлён для кэширования иконок и файла манифеста.

Практическая работа №15 — HTTPS + App Shell

В этой работе приложение было переведено на защищённый протокол HTTPS с использованием локального сертификата, созданного утилитой mkcert. Реализована архитектура App Shell: статический каркас (шапка, навигация) кэшируется и загружается мгновенно, а динамический контент страниц подгружается отдельно по стратегии Network First. Добавлена страница «О приложении» с информацией о проекте.

Практическая работа №16 — WebSocket + Push

В рамках данной работы реализована двусторонняя связь в реальном времени через WebSocket с использованием библиотеки Socket.IO. При добавлении заметки событие мгновенно рассылается всем подключённым клиентам, а во второй вкладке появляется всплывающее уведомление. Также добавлена поддержка Web Push: пользователь может подписаться на push-уведомления и получать их даже при неактивной вкладке.

Практическая работа №17 — Детализация Push

В этой работе функционал уведомлений был расширен: добавлена форма для создания заметки с указанием даты и времени напоминания. Сервер планирует отправку push-уведомления через setTimeout и хранит активные напоминания в памяти. В уведомлении отображается кнопка «Отложить на 5 минут», нажатие на которую переносит напоминание через соответствующий запрос к серверу.
