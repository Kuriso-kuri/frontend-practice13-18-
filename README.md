# Приложение заметок — Практические занятия 13–18

Веб-приложение для управления заметками с поддержкой PWA, WebSocket и Push-уведомлений.

## Функционал

- Добавление заметок и сохранение в `localStorage`
- Архитектура **App Shell** — мгновенная загрузка интерфейса из кэша
- Работа в **офлайн-режиме** через Service Worker
- **HTTPS** с локальным сертификатом (mkcert)
- Установка как нативное приложение (**PWA**)
- Двусторонняя связь в реальном времени через **WebSocket (Socket.IO)**
- **Push-уведомления** при добавлении новой заметки
- **Напоминания** — добавление заметки с датой и временем, push приходит в нужный момент
- Кнопка **«Отложить на 5 минут»** прямо в уведомлении

## Технологии

| Часть | Технологии |
|-------|-----------|
| Frontend | HTML, CSS (Chota), JavaScript, Service Worker |
| Backend | Node.js, Express, Socket.IO, web-push |
| PWA | Web App Manifest, Cache API, Push API |
| Безопасность | HTTPS, VAPID-ключи |

## Запуск

```bash
npm install
node server.js
```

Открыть в браузере: `https://localhost:3001`

> При первом открытии браузер может показать предупреждение о сертификате — нажмите «Дополнительно» → «Перейти на сайт».
